Kết Nối Google Sheets Online Thực Tế

Mặc định trang web lưu trữ dữ liệu trên trình duyệt của riêng bạn. Nếu muốn đồng bộ để nhiều người điền cùng lúc về 1 file Google Sheets duy nhất, hãy làm theo hướng dẫn dưới đây.

Mẹo nâng cao cho Quản trị viên: Dưới đây là đoạn mã Google Apps Script giúp kết nối Form của HTML này trực tiếp với tệp Google Sheets trực tuyến của bạn để nhận dữ liệu thời gian thực miễn phí.

Mã lập trình Google Apps Script (Copy dán vào Tiện ích mở rộng Google Sheets):

function doPost(e) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName("Du_Lieu_Form");
  if (!sheet) {
    sheet = SpreadsheetApp.getActiveSpreadsheet().insertSheet("Du_Lieu_Form");
    sheet.appendRow(["Dấu thời gian", "Họ và tên", "Bộ phận", "Ngày bắt đầu", "Ngày kết thúc", "Số ngày nghỉ", "Lý do"]);
  }
  var data = JSON.parse(e.postData.contents);
  sheet.appendRow([new Date(), data.name, "Văn Phòng", data.startDate, data.endDate, data.days, data.reason]);
  return ContentService.createTextOutput(JSON.stringify({"status": "success"})).setMimeType(ContentService.MimeType.JSON);
}

Khi dán link Apps Script vào đây, đơn phép gửi đi sẽ tự động bắn thẳng về file Google Sheets của bạn!
